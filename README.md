### Plugins

ngCordova:
```sh
$ bower install ngCordova
```

```javascript
<script src="lib/ngCordova/dist/ng-cordova.js"></script>
<script src="cordova.js"></script>
```

Geolocation:
```sh
$ cordova plugin add cordova-plugin-geolocation
```

Compass
```sh
$ cordova plugin add cordova-plugin-device-orientation
```
